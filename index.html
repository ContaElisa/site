Tutorial de Maquiagem

Este tutorial ensina como fazer uma maquiagem básica para o dia a dia.

  1. Prepare a pele com um primer.
  2. Aplique a base em todo o rosto, com movimentos leves.
  3. Corrija as imperfeições com um corretivo.
  4. Aplique o pó para fixar a maquiagem.
  5. Escolha um blush para dar cor às bochechas.
  6. Aplique o iluminador para dar destaque ao rosto.
  7. Aplique o rímel para alongar os cílios.
  8. Aplique o batom para finalizar a maquiagem.

Dicas de maquiagem

  • Escolha produtos que sejam adequados para o seu tipo de pele.
  • Não exagere na maquiagem.
  • Use uma base com FPS para proteger a pele do sol.
  • Remova toda a maquiagem antes de dormir.
